# Python Legacy Reference

This folder contains the original Python transcription script, dependency list, and old `.env.example`.

The current LectureScribe desktop app does not require Python, pip, virtualenv, or this script at runtime. The native app uses the Tauri UI and Rust engine under `lecturescribe-tauri/`.

Keep this archive only as a reference while comparing behavior or migrating older workflows. Do not use the legacy `.env.example` for the current app; API keys are saved through the app setup flow.
